# Global Sales Performance Dashboard

An interactive Excel dashboard for analyzing global sales data from 2023 to 2025. It tracks key performance indicators (KPIs) such as total orders, delivery rates, and regional performance across multiple countries like EGY, KSA, and UAE.

## üìä Key Features
* **Dynamic Slicers:** Filter data by Year (2023-2025), Quarter, Country, and Category (Accessories, Laptop, Mobile).
* **KPI Metrics:** Real-time tracking of #Orders (10,000), Completed Orders (4,542), Pending, and Lost Orders.
* **Visual Analysis:** Monthly order trends and delivery status distribution (45% Delivery Rate).
* **Multi-Currency & Payment Support:** Analyze data in EGP, EUR, USD and view methods like Cash, Credit Card, and PayPal.
